## How to Contribute

1.  **Fork the repository** to your own GitHub account.
2.  **Clone the project** to your machine.
3.  **Create a branch** locally with a succinct name that describes your work.
4.  **Commit changes** to the branch following our commit message standards.
5.  **Push changes** to your fork.
6.  **Open a Pull Request** in our repository and reference any related issues.

## Code of Conduct
Please note that we have a Code of Conduct, and we expect all contributors to adhere to it. Be kind and respectful.
